# Recipes
Android application implemented in Xamarin, for a university project.<br />
The application fetches lists of recipes from spoonacular API according to different keyword enetered by the user. Each desired recipe can be save to a local DB in the phone for a later use.<br />
SRS, design and user-guide documents are attached.<br />
